# Next.js Task Dashboard

A simple Task Management App built with Next.js, TypeScript, Tailwind CSS, and shadcn/ui.

---

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Local state / JSON / Mock API

---

## Setup Steps

Clone the repository:
```bash
git clone https://github.com/ShahanaNazer12/Nextjs-task-dashboard.git

Navigate to the project folder:
cd task-dashboard

Install dependencies:
npm install

Run the development server:
npm run dev

Open http://localhost:3000
